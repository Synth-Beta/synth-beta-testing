# Synth Brand Guide

## Brand Identity
**Synth** is a modern, music-focused community platform that connects people through shared musical experiences. The brand embodies energy, creativity, and the vibrant culture of live music.

## Logo
The Synth logo features:
- **"S"** in bright pink (#FF3399) - represents energy and creativity
- **"ynth"** in light beige (#F5F5DC) - represents sophistication and community
- **Horizontal bar** in light beige - represents connection and unity
- **Crowd silhouette** in black - represents community and shared experiences

## Color Palette

### Primary Colors
- **Synth Pink**: #FF3399
  - Light: #FF66B3
  - Dark: #E6007A
  - Usage: Primary actions, highlights, music category

- **Synth Beige**: #F5F5DC
  - Light: #FAFAF5
  - Dark: #E6E6D1
  - Usage: Secondary actions, text, backgrounds

- **Synth Black**: #000000
  - Light: #1A1A1A
  - Dark: #0D0D0D
  - Usage: Text, borders, dark mode backgrounds

### Supporting Colors
- **White**: #FFFFFF - Clean backgrounds
- **Gray Scale**: Various shades for muted text and borders

## Typography
- **Font Family**: Inter (system fallback: -apple-system, BlinkMacSystemFont, sans-serif)
- **Headings**: Bold weight, tight letter spacing (-0.02em to -0.05em)
- **Body Text**: Regular weight, standard letter spacing

## Design Principles

### 1. Bold & Modern
- High contrast between colors
- Clean, geometric shapes
- Strong visual hierarchy

### 2. Music-Focused
- Energetic color palette
- Dynamic animations and transitions
- Community-oriented imagery

### 3. Accessible
- High contrast ratios
- Clear typography
- Intuitive navigation

## Component Styles

### Cards
- Rounded corners (0.75rem)
- Subtle shadows
- Hover effects with elevation
- Synth gradient backgrounds

### Buttons
- **Primary (Synth)**: Pink background with glow effect
- **Secondary**: Beige background with black text
- **Ghost**: Transparent with hover states

### Navigation
- Bottom navigation bar
- Active states with pink background
- Smooth transitions

### Event Categories
- **Music**: Synth pink theme
- **Food**: Orange theme
- **Arts**: Purple theme
- **Sports**: Green theme
- **Social**: Synth beige theme

## Usage Guidelines

### Do's
- Use the Synth logo prominently in headers
- Apply pink for primary actions and music-related content
- Use beige for secondary elements and text
- Maintain high contrast for accessibility
- Use the crowd silhouette imagery for community features

### Don'ts
- Don't use colors outside the defined palette
- Don't modify the logo colors or proportions
- Don't use low contrast combinations
- Don't overcrowd the interface with too many colors

## Implementation
All brand styles are implemented through:
- Tailwind CSS custom classes
- CSS custom properties
- Component variants
- Design system utilities

## Dark Mode
- Black backgrounds (#000000)
- White text (#FFFFFF)
- Pink accents maintained
- Beige used sparingly for contrast
