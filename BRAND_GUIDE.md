# Brand Guide

This comprehensive brand guide covers the visual identity, design system, and brand guidelines for the PlusOne Event Crew application.

## Brand Identity

**PlusOne** is a modern, music-focused community platform that connects people through shared musical experiences. The brand embodies energy, creativity, and the vibrant culture of live music.

### Brand Values
- **Community**: Bringing people together through music
- **Discovery**: Helping users find new artists and venues
- **Authenticity**: Real reviews from real music lovers
- **Inclusivity**: Welcoming all music tastes and preferences
- **Energy**: Capturing the excitement of live music

## Logo

### Primary Logo
The PlusOne logo features:
- **"PlusOne"** in bold, modern typography
- **Music note icon** integrated with the text
- **Clean, minimalist design** that works across all platforms
- **Scalable vector format** for all use cases

### Logo Variations
- **Full logo**: Complete logo with text and icon
- **Icon only**: Music note icon for favicons and small spaces
- **Text only**: Typography-only version for specific contexts
- **Monochrome**: Single-color versions for special applications

### Logo Usage Guidelines
- **Minimum size**: 24px height for digital use
- **Clear space**: Equal to the height of the "P" in PlusOne
- **Background**: Works on light and dark backgrounds
- **Don't**: Stretch, rotate, or modify the logo

## Color Palette

### Primary Colors

#### PlusOne Pink (#FF3399)
- **Light**: #FF66B3
- **Dark**: #E6007A
- **Usage**: Primary actions, highlights, music category
- **Accessibility**: Meets WCAG AA contrast requirements

#### PlusOne Beige (#F5F5DC)
- **Light**: #FAFAF5
- **Dark**: #E6E6D1
- **Usage**: Secondary actions, text, backgrounds
- **Accessibility**: High contrast with dark text

#### PlusOne Black (#000000)
- **Light**: #1A1A1A
- **Dark**: #0D0D0D
- **Usage**: Text, borders, dark mode backgrounds
- **Accessibility**: Maximum contrast for readability

### Secondary Colors

#### Success Green (#10B981)
- **Usage**: Success states, positive actions
- **Accessibility**: High contrast for important information

#### Warning Orange (#F59E0B)
- **Usage**: Warnings, attention-grabbing elements
- **Accessibility**: Clear visibility for alerts

#### Error Red (#EF4444)
- **Usage**: Error states, destructive actions
- **Accessibility**: High contrast for critical information

#### Info Blue (#3B82F6)
- **Usage**: Information, links, neutral actions
- **Accessibility**: Good contrast for secondary information

### Color Usage Guidelines
- **Primary actions**: Use PlusOne Pink
- **Secondary actions**: Use PlusOne Beige
- **Text**: Use PlusOne Black on light backgrounds
- **Backgrounds**: Use PlusOne Beige for main backgrounds
- **Accents**: Use secondary colors sparingly for specific states

## Typography

### Font Family
- **Primary**: Inter (Google Fonts)
- **Fallback**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
- **Monospace**: "SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, monospace

### Font Weights
- **Light**: 300 (for large headings)
- **Regular**: 400 (for body text)
- **Medium**: 500 (for emphasis)
- **SemiBold**: 600 (for subheadings)
- **Bold**: 700 (for headings)

### Typography Scale
- **H1**: 2.5rem (40px) / Bold / Line height 1.2
- **H2**: 2rem (32px) / SemiBold / Line height 1.3
- **H3**: 1.5rem (24px) / SemiBold / Line height 1.4
- **H4**: 1.25rem (20px) / Medium / Line height 1.4
- **H5**: 1.125rem (18px) / Medium / Line height 1.5
- **H6**: 1rem (16px) / Medium / Line height 1.5
- **Body**: 1rem (16px) / Regular / Line height 1.6
- **Small**: 0.875rem (14px) / Regular / Line height 1.5
- **Caption**: 0.75rem (12px) / Regular / Line height 1.4

### Typography Guidelines
- **Headings**: Use hierarchy to create clear information structure
- **Body text**: Maintain good line height for readability
- **Links**: Use PlusOne Pink with hover states
- **Code**: Use monospace font for technical content

## Iconography

### Icon Style
- **Style**: Outline icons with 2px stroke width
- **Size**: 24px for standard use, 16px for small contexts
- **Color**: Inherit from parent element
- **Library**: Heroicons or similar outline icon set

### Common Icons
- **Music**: Musical note icon
- **Events**: Calendar icon
- **Users**: User group icon
- **Search**: Magnifying glass icon
- **Heart**: Heart icon for likes
- **Star**: Star icon for ratings
- **Chat**: Chat bubble icon
- **Settings**: Cog icon

## Layout Guidelines

### Grid System
- **Container**: Max width 1200px, centered
- **Columns**: 12-column grid system
- **Gutters**: 24px between columns
- **Breakpoints**: Mobile (320px), Tablet (768px), Desktop (1024px)

### Responsive Design
- **Mobile-first**: Design for mobile, enhance for larger screens
- **Breakpoints**: Use consistent breakpoints across components
- **Touch targets**: Minimum 44px for touch interactions
- **Content**: Prioritize content hierarchy on small screens

## Accessibility

### Color Contrast
- **Text on background**: Minimum 4.5:1 contrast ratio
- **Large text**: Minimum 3:1 contrast ratio
- **Interactive elements**: Minimum 3:1 contrast ratio

### Focus States
- **Visible focus**: Clear focus indicators for keyboard navigation
- **Focus order**: Logical tab order through interface
- **Skip links**: Provide skip navigation for screen readers

### Screen Reader Support
- **Alt text**: Descriptive alt text for all images
- **ARIA labels**: Proper ARIA labels for interactive elements
- **Semantic HTML**: Use proper HTML elements for content structure

## Dark Mode

### Dark Mode Colors
- **Background**: #0D0D0D (PlusOne Black Dark)

- **Text**: #F5F5DC (PlusOne Beige)
- **Accent**: #FF3399 (PlusOne Pink)
- **Borders**: #374151 (Gray 700)

### Dark Mode Guidelines
- **Consistent contrast**: Maintain accessibility standards
- **Color meaning**: Preserve color meaning across themes
- **Smooth transitions**: Animate theme changes
- **User preference**: Respect system preference

## Brand Voice & Tone

### Voice Characteristics
- **Friendly**: Approachable and welcoming
- **Enthusiastic**: Excited about music and community
- **Authentic**: Genuine and honest
- **Inclusive**: Welcoming to all music tastes

### Tone Guidelines
- **Headlines**: Energetic and exciting
- **Body text**: Clear and helpful
- **Error messages**: Supportive and solution-oriented
- **Success messages**: Celebratory and encouraging

### Writing Style
- **Active voice**: Use active voice when possible
- **Concise**: Clear and to the point
- **Conversational**: Natural and human
- **Inclusive**: Use inclusive language
